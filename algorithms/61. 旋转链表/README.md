# [61. 旋转链表](https://leetcode-cn.com/problems/rotate-list/)


## 题目大意
给你一个链表的头节点 head ，旋转链表，将链表每个节点向右移动 k 个位置。

 

示例 1：


输入：head = [1,2,3,4,5], k = 2
输出：[4,5,1,2,3]
示例 2：


输入：head = [0,1,2], k = 4
输出：[2,0,1]
 

提示：

链表中节点的数目在范围 [0, 500] 内
-100 <= Node.val <= 100
0 <= k <= 2 * 109

## 解题思路
先统计链表元素总数
然后 k % 总数 = 到K的位置
然后把尾巴与头部相连 形成一个环
然后找到断开的位置。切断环 返回数据
